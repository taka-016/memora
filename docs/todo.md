## 旅行管理画面

- 旅程の編集時に場所を指定できるようにする
  - 場所未指定の場合は「場所を指定」ボタンを表示する
  - 「場所を指定」ボタンタップでボタンの下に`google_map_view`を使ったマップ画面を表示する
  - マップの初期表示位置は、①場所が指定されている場合はそのlocationの位置、②場所が未指定で、旅行に1件でもlocationが紐づいている場合は紐づく1件目の位置、③旅行にもlocationが紐づいていない場合は、初期位置(または現在地)
  - 既に場所が指定されている場合は、「場所を変更」「指定を解除」の2つのボタンに変え、下に紐付くlocationの場所名を表示
  - マップ上の長押しで新しいlocationを設定する
  - 場所検索でlocationを設定することも可能
  - 旅行に紐づく既存locationsを灰色ピンで表示する
  - 灰色ピンをタップすると場所名と「この場所を指定する」を表示し、タップしたlocationの`locationId`を旅程に設定する
  - その旅程に紐づくlocationのピンだけ赤色で表示する
  - 赤色ピンをタップすると旅程との紐付けを解除できるようにする
- 旅行編集画面の下部に旅行のlocationsマップを表示する
  - 旅程ボタンとタスクボタンの下に`google_map_view`を表示する
  - マップの拡大ボタンを押すと、マップだけの画面を表示する
  - この旅行に紐づくlocationsを赤色ピンで表示する
  - ピンをタップすると紐づく旅程名を表示する
  - ピンをタップした場所からlocationを削除できるようにする
  - マップの長押しで旅行に紐づくlocationを追加できるようにする
  - 長押しで追加したlocationは旅程に紐づけず、旅行にのみ紐づくlocationとして保存する
